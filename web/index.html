¿Por qué en mi navegador no se ve en todos los simbolos, en ocaciones cuadrados?
Los símbolos están contenidos al igual que las letras y números en las fuentes, que son pequeños paquetes que contienen caracteres.
Windows incluye varios tipos de fuentes, pero cada navegador de forma predeterminada carga el tipo de fuentes para el que está diseñado y busca los símbolos dentro de ellas, lógicamente las fuentes no pueden incluir todo tipo de símbolos, sino serían muy pesadas.
Algunos ejemplos:
✔ El navegador Internet Explorer utiliza la fuente: Arial
✔ Firefox utiliza: MS Reference Serif
✔ Google Chrome: Times New Roman
Para solucionar este problema solo carga la página en un navegador diferente. El navegador donde se representan mejor todos los símbolos y caracteres especiales es Firefox.
